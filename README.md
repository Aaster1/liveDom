## 주제
<hr>
<ul>
  <li>
    공연밴드팀과 대관서비스 제공자, 공연티켓 구매자들을 위한 웹 사이트 커뮤니티
  </li>
</ul>

## 프로젝트 배경
<hr>
<ul>
  <li>해당 서비스를 제공하는 기존의 국내 사이트, 『뮬』은 오래된 사이트라 디자인에 대한 평이 상당히 좋지 않음.</li>
  <li>대관, 공연팀 모집 등 서비스에서 직접 연락을 통해 진행해야하는 점이 불편함.</li>
  <li>구글폼이나 외부 사이트를 이용하여 관리 과정이 복잡함.</li>
</ul>
